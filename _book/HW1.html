  1. What do we learn from the figures in question 3?

From the figures in question 3, it becomes easy to learn that the model predictions vary significantly across the five different models (F1 to F5) when x1 ranges from -3 to 3, with x2 held constant at -0.5 and x3 set to either 0 or 1. These predictions varies uniquely for each black model value from F1-F5 and x3 values changing and with these variations we can gain an understanding how each model creates different outcomes and graph/model behaviors regardless of utilizing the same input variable x1 from -3 to 3 with incrementation at the value of 0.5. Altogether, this figure helps us understand that there will always be differences in predictions and we need a model by model comparison to better understand these predictions and what is does to the interpretations and decisions we make based on these models.
